# ContentFlow - SaaS de Génération de Contenu

Une app que tu peux vendre pour générer 30 jours de contenu (LinkedIn, TikTok, Email) en 2 minutes.

## Fichiers clés:

- **`pages/index.js`** - Interface utilisateur (React)
- **`pages/api/generate.js`** - API qui appelle Claude
- **`DEPLOYMENT.md`** - Guide complet de déploiement

## Quick start:

```bash
# 1. Clone/copie les fichiers
# 2. npm install
# 3. Crée .env.local avec ta clé Anthropic
# 4. npm run dev
# 5. Visite http://localhost:3000
```

## Déployer:

Voir `DEPLOYMENT.md` pour les instructions complètes.
